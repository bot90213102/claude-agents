## 📖 项目简介

Claude Agents 是一个精心设计的AI代理角色库，包含 **34个专业化的AI助手配置文件**。每个代理都针对特定的职业角色和工作场景进行了优化，帮助用户在各个领域提升工作效率和质量。

## 🎯 核心特性

- **🔧 工程技术** - 从前端开发到DevOps自动化的完整技术栈
- **📱 产品管理** - 趋势研究、用户反馈分析、敏捷开发支持  
- **📈 数字营销** - 全平台社交媒体策略与内容优化
- **🎨 设计创意** - UI/UX设计、品牌管理、视觉叙事
- **📋 项目管理** - 实验跟踪、项目交付、工作室运营
- **🔧 运营支持** - 基础设施维护、合规检查、数据分析
- **🧪 质量测试** - 性能基准测试、API测试、工作流优化

## 🚀 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/bot90213102/claude-agents.git
cd claude-agents
```

### 2. 选择代理

1. 浏览 `agents/` 目录下对应领域的文件夹
2. 选择符合需求的代理角色
3. 复制对应的 `.md` 配置文件内容

### 3. 使用代理

将选择的代理配置文件内容复制到 Claude 对话中，即可开始使用专业化的AI助手。

## 📋 代理特性

每个代理都包含以下标准化结构：

- **🎯 角色定位** - 清晰的专业身份和背景
- **🛠️ 核心技能** - 相关工具和专业能力清单
- **📋 主要任务** - 明确的职责范围和目标
- **🔄 工作流程** - 7步标准化工作流程
- **💬 使用指示** - 具体的交互指导和使用说明
